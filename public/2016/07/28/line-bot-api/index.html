註冊申請Line 機器人

  • Line developer 官網 申請機器人帳號
  • 申請註冊channel後，設定Callback URL，必需要是https而且不能是自行簽署的憑證，還要符合line認可的CA list
  • 還需在Server IP Whitelist 加上你提供callback url的主機ip
  • 取得API需使用的三組代碼Channel ID Channel Secret MID
