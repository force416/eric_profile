在github上架設hexo

這篇是在講如何在github的靜態頁面上架設hexo blog framework

申請github帳號

install nodejs

  • 上nodejs的官網下載並安裝

npm install hexo

  • 上hexo的官網有安裝教學

在github上建立repositories

  • 建立的專案名稱eric_profile

將專案發佈上github page

  • 到該專案的settings設定github pages點選launch automatic page generaotr 一路照預設的點next
  • 建立完後會建立 branch 為gh-pages，只有在這個branch下的才會在github pages上顯示，網址是https://username.github.io/eric_profile/

設定hexo deploy to github

  • 修改 _config.yml 設定deploy git 資訊

產生檔案 & deploy
