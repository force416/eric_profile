在ubuntu上安裝mysql db

這篇是描述如何在ubuntu上安裝mysql的過程

透過 apt-get 安裝

安裝 mysql_secure_installation

調整可連線的IP

  • 一般預設都是只能給本機(127.0.0.1)連線，如果要給外部的ip連線則需修改設定檔

注意事項

  • 安裝完mysql預設的port是3306
  • 如果ubuntu是aws 的 ec2 則需注意security group是否有開起mysql的port 3306
