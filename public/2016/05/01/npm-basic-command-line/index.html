npm 基本指令

npm 簡介

  • npm 全名為 Node Package Manager，是 Node.js 的套件（package）管理工具

  • 產生package.json 檔案

  • 如果專案裡已有package.json可以直接執行 install

  • 全域安裝加上-g的參數

  • 專案安裝不加-g的參數

  • 專案安裝並加入package.json檔案裡，需加上--save的參數
